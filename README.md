Vue.js project for Yocto image

Run 'npm install' after cloning to get started 

Develop on host by editing files in /src and running 'npm run dev'

When you are ready run 'npm run build', do 'git add . && git commit && git push' 
The next time you build the yocto image the changes will be included

Or you can copy the data in /dist to /var/www/html
